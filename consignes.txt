Après clonage du projet depuis github :

1. Installer les modules à l'aide de la commande : npm install
2. créer le dossier setting dans lequel sera créé le fichier privateKey.js
3. Dans le fichier privatekey.js, créer la clé privée qui sera utilisée pour la création et la vérification du token.
    Ecrire la ligne suivante : module.exports = "*"
    (où * représente une chaine de caractères de votre choix)
